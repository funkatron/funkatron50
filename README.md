# Funkatron 50

> Fifty songs that helped make me who I am.

This repository is the working source for the Funkatron 50 project.

The goal is not to rank songs, but to preserve the stories, discoveries, and connections behind fifty songs that had a lasting impact on my life.

## Repository layout

- `songs/` — one Markdown file per song
- `playlists/` — canonical playlist and genre lists
- `notes/` — observations, recurring themes, and research

We'll build this one song at a time.
